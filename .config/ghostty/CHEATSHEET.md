# Ghostty Cheat Sheet

## Tabs

| Key | Action |
|---|---|
| `ctrl+shift+t` | New tab |
| `ctrl+shift+w` | Close tab |
| `ctrl+tab` | Next tab |
| `ctrl+shift+tab` | Previous tab |
| `alt+1` … `alt+8` | Jump to tab N |
| `alt+9` | Last tab |

## Splits

| Key | Action |
|---|---|
| `ctrl+shift+o` | Split right |
| `ctrl+shift+e` | Split down |
| `ctrl+alt+arrow` | Focus split |
| `super+ctrl+shift+arrow` | Resize split (10 px) |
| `ctrl+shift+enter` | Zoom / unzoom split |
